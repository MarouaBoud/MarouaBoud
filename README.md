## Professional Project Summaries

###  **Stealth Power – Data & Machine Learning Engineer**  
 *May 2024 – Present (Austin, TX – Remote)*  
 *Python, AWS, Google Cloud, PostgreSQL, Streamlit, XGBoost, Deep Learning, Node.js, Git, Bash, UML, RESTful APIs, MLflow*

**Overview:**  
At Stealth Power, I designed and deployed **machine learning and data systems** to optimize solar energy production and performance.  
My work focused on building **real-time data pipelines**, **forecasting models**, and **AI-powered applications** for intelligent energy monitoring.

**Key Contributions:**  
- Architected and deployed a **real-time alert system** for solar energy monitoring, including RESTful API design, data pipelines, and Streamlit dashboard integration.  
- Built **interactive dashboards** for live tracking of solar metrics and stakeholder reporting.  
- Developed a full **end-to-end ML pipeline** for solar energy production and consumption forecasting:  
  - Data ingestion, cleaning, and feature engineering from multiple heterogeneous sources.  
  - Training of a **multi-output deep learning model** for production and consumption forecasting.  
  - Deployment of the model as a **REST API** with an **SDK** for seamless integration into production systems.  
- Designed and implemented **AI Agents and RAGs (Retrieval-Augmented Generation systems)** using solar system data — enabling customers to query, interpret, and visualize system performance insights using natural language.  
- Collaborated with cross-functional teams to bring **data-driven insights** into production-grade tools and client dashboards.
